    login() click to toggle source

    Saves logged admin to session if login and password correct, and redirect to members list page. Add error to @error if incorrect username or password, and renders login form.

    # File app/controllers/admins_controller.rb, line 17
def login
  admin = Admin.find_by_login(params[:login]).try(:authenticate, params[:password])
  @error = Hash.new
  if  admin
    session[:admin] = admin
    redirect_to members_path and return
  end
  @error[:error] = "Virheellinen kayttajatunnus tai salasana"
  render "loginform"
end
